A modern, mobile-first, responsive blog-style site built with HTML, CSS, and JavaScript.

Features
- Sticky header with navigation, social icons, and subscribe button
- Horizontal top stories ticker with smooth auto-scroll and pause-on-hover
- Responsive card grid: 2 (mobile), 3 (tablet), 4 (desktop) cards per row
- Cards have image thumbnails, category badges, show prompt toggle, author, hover lift, and equal-height layout
- Theme toggle (Light / Night / Dark) with persistent preference
- Individual post pages for each prompt with copy-to-clipboard button
- Sticky sidebar on desktop with Search, Recent posts, Comments, and Categories
- Lazy-loaded images and accessible controls

How to run
1. Open `index.html` in your browser.
2. No build step required.

Notes
- Uses pure CSS and vanilla JavaScript. Replace placeholder images with real thumbnails if you want to use real content.
